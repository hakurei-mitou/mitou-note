# IntelliJ IDEA

## 阅读快捷键

- `Alt + <-` 

	返回上一个文件位置。（粒度大）

- `Ctrl + Alt + <-` 

	返回上一个光标位置。（粒度小）

## 输入快捷键

- 快速接收返回值

	输入函数后按 `Alt + Enter` 。

## 多光标编辑

IDEA 里的 IdeaVIM 插件的 Ctrl + V（块选择模式）不能插入，可用多光标编辑替代：

- 连续行选中

	按住 Alt + 鼠标左键滑动（可通过 Ctrl + right 等方式进一步控制各光标的位置）。

- 多处选中

	按住 Ctrl + Shift + Alt + 左键点击。
